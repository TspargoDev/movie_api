API Documentation

Movies API

Request URL Response
GET /api/movies Returns a list of all movies.
GET /api/movies/:id Returns details of a specific movie by its ID.
POST /api/movies Adds a new movie to the database. Expects movie data in the request body.
PUT /api/movies/:id Updates an existing movie's details by its ID. Expects updated movie data in the request body.
DELETE /api/movies/:id Deletes a specific movie by its ID.

Users API

Request URL Response
